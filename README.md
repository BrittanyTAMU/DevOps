**Too Lazy To Use the Calculator On Your Phone ehh? Use Mine :)**



The calculator website project utilizes JavaScript Object-Oriented Programming (OOP) best practices to create two distinct calculators, one mimicking the iPhone calculator interface and the other resembling a traditional calculator, providing users with intuitive and functional tools for mathematical calculations(addition, subtraction, multiplication and division).

**How It's Made:**
Tech used: HTML, CSS, JavaScript.

The calculator website is built using HTML for the structure, CSS for styling, and JavaScript for functionality. Object-oriented programming (OOP) principles are applied in the JavaScript code to create modular and reusable components for the calculators. Event listeners are utilized to capture user input, and the DOM (Document Object Model) is manipulated dynamically to update the calculator display accordingly. Additionally, CSS is used to ensure the calculators have a visually appealing and responsive design across different devices.

**Lessons Learned:**
**User-Centric Approach**: Optimizing the project underscores the importance of considering the end user's experience. Prioritizing optimizations that enhance usability, accessibility, and overall satisfaction reinforces the significance of a user-centric design approach.
